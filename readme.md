# SQLite Korean Dictionaries

This repository contains a JSON to SQLite converter script & prebuilt SQLite dictionaries using the [KRDict](https://krdict.korean.go.kr/) dictionary from the Korean Language Institute. The original JSON extractions are made by [RicBent](https://github.com/RicBent/KRDict-Converter/releases/tag/2.1.0)

RUN: node buildDictionaryDB.js
